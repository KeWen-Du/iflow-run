## 项目概述

**iFlow-run** 是一个用于查看 iFlow CLI 会话轨迹和历史会话的 Web 应用程序。它提供了可视化的界面来浏览、搜索和分析 iFlow CLI 的交互历史。

### 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Node.js + Express + TypeScript |
| 前端 | 纯 HTML5 + CSS3 + JavaScript |
| 样式 | 自定义 CSS，暗色主题 |
| 实时通信 | WebSocket |
| 构建 | Vite |
| 模块系统 | CommonJS |

### 核心价值

- 📊 **可视化**：将 iFlow CLI 的 JSONL 会话文件转换为易读的 Web 界面
- 🔍 **可搜索**：快速搜索和筛选历史会话
- 📈 **统计分析**：Token 消耗、工具使用统计
- 🔄 **实时更新**：WebSocket 监听文件变更

| 📁 项目管理 | 浏览和查看 iFlow CLI 创建的所有项目 | v1.0.0 |

| 🔍 消息详情 | 查看完整的对话消息（用户/助手/工具调用/工具结果） | v1.0.0 |

### 其他功能

- 🔎 消息筛选：按类型筛选（用户/助手/工具调用）和内容搜索
- 💰 Token 统计：模型名称、Token 消耗、执行时间、预估成本
- 🔄 环境追踪：检测工作目录和 Git 分支变更
- 📥 导出功能：Markdown 或 JSON 格式导出
- 📄 分页加载：项目列表分页，消息分页加载
- 🎨 Markdown 渲染：支持代码语法高亮
- 🚀 后台运行：支持 daemon 模式
- 🔌 端口自动检测：自动选择可用端口

## 新功能提议

### 🏷️ 会话标签系统

**优先级**：🔴 高

**描述**：
- 支持用户自定义标签分类管理会话
- 智能标签：自动识别项目类型、技术栈（如 React、Python、TypeScript）
- 标签筛选：快速按标签过滤会话
- 标签统计：展示各标签的会话数量

**实现思路**：
1. 前端：标签选择器组件、标签筛选 UI
2. 后端：新增标签字段到会话元数据
3. 存储：localStorage 或后端 JSON 文件

**预期收益**：提升会话管理效率，支持项目类型分类
